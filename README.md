# Profiling
Inferring Private Personal Attributes of Virtual Reality Users from Head and Hand Motion Data

The main machine learning code is in `model.py`. For more information, see the [paper](https://arxiv.org/abs/2305.19198) or [website](https://rdi.berkeley.edu/metaverse/profiling/).

Dependencies:
- pandas
- numpy
- pytorch
- tqdm
- sklearn
